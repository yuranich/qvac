# QVAC NMT Addon v0.3.8 Release Notes

This release introduces the new `addon-cpp` integration for the NMT addon and prepares the package for distribution as v0.3.8. The integration improves how native addon components are connected in the build and runtime path, making C++ addon usage more consistent across environments.

## Features

### `addon-cpp` Integration

The addon now includes integration with `addon-cpp`, enabling a cleaner and more reliable native integration layer for the NMT pipeline. This improves maintainability of the native boundary and helps reduce platform-specific integration friction.

## Other

### Package Version Update

The package version has been updated from 0.3.7 to 0.3.8 in `package.json` so this release can be published and tracked independently.
