# v0.1.2

## New Features

- **Unified transcription script**: Consolidated `es-transcribe.js`, `fr-transcribe.js`, and `hr-transcribe.js` into a single `transcribe.js` with CLI flags (`--file`, `--model`) for flexible multilingual transcription

## Improvements

- **C++ code modernization**: Replaced multiple `if` status checks with `std::ranges::find` for cleaner code in `Addon.cpp`
- **Code refactoring**: Extracted `computeFeatures()` and `runInferencePipeline()` helper functions in `ParakeetModel.cpp` for better modularity and maintainability

## Documentation

- Updated `README.md` and `QUICKSTART.md` with new `transcribe.js` usage examples
- Updated `download-models-s3.sh` post-download instructions to reference new transcription script

## Breaking Changes

- Removed individual language transcription scripts:
  - `examples/es-transcribe.js`
  - `examples/fr-transcribe.js`  
  - `examples/hr-transcribe.js`
  
  Use `examples/transcribe.js --file <audio-file>` instead.
