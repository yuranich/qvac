## Overview

`@qvac/sdk` npm package exposes a function-centric, typed JS API.

## Functions

| Function | Summary | Signature |
| --- | --- | --- |
| [`cancel()`](/v0.6.1/sdk/api/cancel) | Cancels an ongoing operation. | `cancel(params): Promise<void>` |
| [`close()`](/v0.6.1/sdk/api/close) | Closes the RPC connection to the QVAC server. | `close(): Promise<void>` |
| [`completion()`](/v0.6.1/sdk/api/completion) | Generates completion from a language model based on conversation history. | `completion(params): { tokenStream, toolCallStream, text, toolCalls, stats }` |
| [`defineHandler()`](/v0.6.1/sdk/api/defineHandler) | Helper function to define a handler with full type inference. | `defineHandler<TRequest, TResponse>(definition): PluginHandlerDefinition` |
| [`definePlugin()`](/v0.6.1/sdk/api/definePlugin) | Helper function to define a plugin with full type inference. | `definePlugin<T>(plugin): T` |
| [`deleteCache()`](/v0.6.1/sdk/api/deleteCache) | Deletes KV cache files. | `deleteCache(params): Promise<{ success: boolean }>` |
| [`downloadAsset()`](/v0.6.1/sdk/api/downloadAsset) | Downloads an asset (model file) without loading it into memory. | `downloadAsset(options): Promise<string>` |
| [`embed()`](/v0.6.1/sdk/api/embed) | Generates embeddings for a single text using a specified model. | `embed(params): Promise<number[]>` |
| [`getLogger()`](/v0.6.1/sdk/api/getLogger) | Creates or retrieves a cached logger instance for the given namespace. | `getLogger(namespace, options?): Logger` |
| [`getModelByName()`](/v0.6.1/sdk/api/getModelByName) | Retrieves a model constant from the built-in registry by its human-readable name. | `getModelByName(name): RegistryItem \| undefined` |
| [`getModelBySrc()`](/v0.6.1/sdk/api/getModelBySrc) | Retrieves a model constant from the built-in registry by its model ID and blob core key. | `getModelBySrc(modelId, blobCoreKey): RegistryItem \| undefined` |
| [`getModelInfo()`](/v0.6.1/sdk/api/getModelInfo) | Retrieves detailed information about a model, including cache status and loaded instances. | `getModelInfo(params): Promise<ModelInfo>` |
| [`invokePlugin()`](/v0.6.1/sdk/api/invokePlugin) | Invoke a non-streaming plugin handler. | `invokePlugin<TResponse, TParams>(options): Promise<TResponse>` |
| [`invokePluginStream()`](/v0.6.1/sdk/api/invokePluginStream) | Invoke a streaming plugin handler. | `invokePluginStream<TResponse, TParams>(options): AsyncGenerator<TResponse>` |
| [`loadModel()`](/v0.6.1/sdk/api/loadModel) | Loads a machine learning model from a local path, remote URL, or Hyperdrive key. | `loadModel(options): Promise<string>` |
| [`loggingStream()`](/v0.6.1/sdk/api/loggingStream) | Opens a logging stream to receive real-time logs. | `loggingStream(params): AsyncGenerator<LoggingStreamResponse>` |
| [`modelRegistryGetModel()`](/v0.6.1/sdk/api/modelRegistryGetModel) | Retrieves a single model entry from the QVAC model registry by path and source. | `modelRegistryGetModel(registryPath, registrySource): Promise<ModelRegistryEntry>` |
| [`modelRegistryList()`](/v0.6.1/sdk/api/modelRegistryList) | Lists all available models from the QVAC model registry. | `modelRegistryList(): Promise<ModelRegistryEntry[]>` |
| [`modelRegistrySearch()`](/v0.6.1/sdk/api/modelRegistrySearch) | Searches the QVAC model registry with optional filters. | `modelRegistrySearch(params?): Promise<ModelRegistryEntry[]>` |
| [`ocr()`](/v0.6.1/sdk/api/ocr) | Performs Optical Character Recognition (OCR) on an image to extract text. | `ocr(params): { blockStream, blocks, stats }` |
| [`ping()`](/v0.6.1/sdk/api/ping) | Sends a ping request to the server and returns the pong response. | `ping(): Promise<{ type: "pong", number: number }>` |
| [`ragChunk()`](/v0.6.1/sdk/api/ragChunk) | Chunks documents into smaller pieces for embedding. | `ragChunk(params): Promise<RagDoc[]>` |
| [`ragCloseWorkspace()`](/v0.6.1/sdk/api/ragCloseWorkspace) | Closes a RAG workspace, releasing in-memory resources (Corestore, HyperDB adapter, RAG instance). | `ragCloseWorkspace(params?): Promise<void>` |
| [`ragDeleteEmbeddings()`](/v0.6.1/sdk/api/ragDeleteEmbeddings) | Deletes document embeddings from the RAG vector database. | `ragDeleteEmbeddings(params): Promise<void>` |
| [`ragDeleteWorkspace()`](/v0.6.1/sdk/api/ragDeleteWorkspace) | Deletes a RAG workspace and all its data. | `ragDeleteWorkspace(params): Promise<void>` |
| [`ragIngest()`](/v0.6.1/sdk/api/ragIngest) | Ingests documents into the RAG vector database. | `ragIngest(params): Promise<{ processed, droppedIndices }>` |
| [`ragListWorkspaces()`](/v0.6.1/sdk/api/ragListWorkspaces) | Lists all RAG workspaces with their open status. | `ragListWorkspaces(): Promise<RagWorkspaceInfo[]>` |
| [`ragReindex()`](/v0.6.1/sdk/api/ragReindex) | Reindexes the RAG database to optimize search performance. | `ragReindex(params): Promise<RagReindexResult>` |
| [`ragSaveEmbeddings()`](/v0.6.1/sdk/api/ragSaveEmbeddings) | Saves pre-embedded documents to the RAG vector database. | `ragSaveEmbeddings(params): Promise<RagSaveEmbeddingsResult[]>` |
| [`ragSearch()`](/v0.6.1/sdk/api/ragSearch) | Searches for similar documents in the RAG vector database. | `ragSearch(params): Promise<RagSearchResult[]>` |
| [`startQVACProvider()`](/v0.6.1/sdk/api/startQVACProvider) | Starts a provider service that offers QVAC capabilities to remote peers. | `startQVACProvider(params): Promise<{ success, publicKey? }>` |
| [`stopQVACProvider()`](/v0.6.1/sdk/api/stopQVACProvider) | Stops a running provider service and leaves the specified topic. | `stopQVACProvider(params): Promise<{ success }>` |
| [`textToSpeech()`](/v0.6.1/sdk/api/textToSpeech) | Converts text to speech audio. | `textToSpeech(params): { bufferStream, buffer, done }` |
| [`transcribe()`](/v0.6.1/sdk/api/transcribe) | Provides a simple interface for transcribing audio by collecting all streaming results into a single string response. | `transcribe(params): Promise<string>` |
| [`transcribeStream()`](/v0.6.1/sdk/api/transcribeStream) | Streams audio transcription results in real-time, yielding text chunks as they become available from the model. | `transcribeStream(params): AsyncGenerator<string>` |
| [`translate()`](/v0.6.1/sdk/api/translate) | Translates text from one language to another using a specified translation model. | `translate(params): { tokenStream, text, stats }` |
| [`unloadModel()`](/v0.6.1/sdk/api/unloadModel) | Unloads a previously loaded model from the server. | `unloadModel(params): Promise<void>` |
